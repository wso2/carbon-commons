Users and Roles

The GUI of the WSO2 Carbon facilitates the management and control of user accounts and roles at different levels. The key functionalities of this component include:

  • Change password by user
  • Add, modify or remove user accounts (Only visible to admins with permissions)
  • Add, modify or remove user roles (Only visible to admins with permissions)

Figure 1: User/Role Management

Change My Password

Any user who has login permission can change their password by clicking on "Change My Password" link. It will require the current password inorder to change it. If you have forgotten your current password please contact your administrator who can reset it without the current password.

Managing Users

Users link is only visible to users with "admin" permission. Refer the context sensitive help available on Users page to add new user accounts, modify existing accounts, or delete existing accounts.

Managing Role

Roles link is only visible to users with "admin" permission. Refer the context sensitive help available on Roles page to add new roles, modify roles and their permissions.
