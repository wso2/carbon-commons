Reports

First you need to create a report template according to your report structure. You can use a report template designer(e.g iReport) to generate this templates.

  1. Using your own report design - Custom Template



    1. Add your Custom report template.
    2. In the navigation under Tools click Reports -> Add -> Custom Report
    3. Add Report Template page appears.

      4. Figure 1. Add Report Template page

    4. Click on Reports -> List , after you will see the available reports list
    5. Reports list page appears.

      6. Figure 2. Reports List

    6. If you want to update the template ,click on the Edit icon.
    7. Report Editor page appears.

      8. Figure 3. Report Editor

    8. Click on the report name , then you will see the report generation page.
    9. Select Data Source and report Type. (If you haven't configured a data source please follow the the guide to create a new Data Source )
    10. Generate Report page appears.

      11. Figure 4. Generate Report

    11. Click on the Generate Report button to generate the report.



  2. Using Default Report Templates

    3. You can also use the report templates which are inbuilt with WSO2 server, and follow the instructions provided in the UI. Follow instruction given Reports using report templates to add reports using templates.
