Using Report Templates

  1. Table Report

    1. In the navigation under Tools click Reports -> Add -> Table Report
    2. Select Data Source and report Type. (If you haven't configured a data source please follow the the guide to create a new Data Source )
    3. Fill all the required fields according to your preference. Once you have completed all the necessary fields the Next button will be activated
    4. Click on Next button, and provide your table report formatting details.
    5. Click on create button which will create the .jrxml file for your report.
    6. Reports list page appears.
    7. If you want to update the template ,click on the Edit icon.
    8. Report Editor page appears.
    9. Click on the report name , then you will see the report generation page.
    10. Select your preferred report type such as pdf, xls, etc
    11. Click on the Generate Report button to generate the report.
    12. This will generate the report

  2. Bar Chart Report

    1. In the navigation under Tools click Reports -> Add -> Bar Chart Report
    2. Select Data Source and report Type. (If you haven't configured a data source please follow the the guide to create a new Data Source )
    3. Fill all the required fields according to your preference. Once you have completed all the necessary fields the Next button will be activated
    4. You can add any number of series you preferred to add. But of course you should take care if the visibility of the number of series in the final chart
    5. Provide the chart axis fields

      • X-Axis - Any String value is permitted. Note even though you have selected a number field , it will be treated as string field
      • Y-Axis - Any Numerical value filed should be selected

    6. Click on Next button, and provide your table report formatting details.
    7. Click on create button which will create the .jrxml file for your report.
    8. Reports list page appears.

  11. Composite Chart Report
