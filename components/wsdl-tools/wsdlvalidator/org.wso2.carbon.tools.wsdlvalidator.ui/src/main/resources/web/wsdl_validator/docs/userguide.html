WSDL Validator

WSDL Validator tool can be used to validate a WSDL document. You can upload your WSDL or you can provide a WSDL URL. The tool will validate your WSDL and print the result in the "Validation Result" section.

Uploading a WSDL

You can upload your WSDL document to be validated, using the following UI.

Figure 1: Uploading a WSDL

Click on "Validate From File" button after selecting the WSDL location.

Providing a WSDL URL

You can provide the URL of your WSDL document to be validated, using the following UI.

Figure 2: Providing a WSDL URL

Click on "Validate From URL" button after providing the WSDL URL.
